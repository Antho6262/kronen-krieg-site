INITIALISATION
Usage unique — Kronen Krieg

Ce bouton crée la base vierge dans Firebase : un seul membre (toi, en admin) avec accès total, et toutes les autres données (stock, actions, semaines, membres suivants…) restent vides pour être configurées depuis l'Admin.

⚠️ Des données existent déjà dans cette base (membres trouvés). Relancer l'initialisation écrasera ce compte mais ne touchera pas le reste.
